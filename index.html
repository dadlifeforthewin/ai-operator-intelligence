Operator Playbook

Best current agent workflow

  1. Write a tight task spec.
  2. Create a fresh branch or git worktree.
  3. Run one focused agent per task.
  4. Give scoped context, not everything.
  5. Require tests, lint, typecheck, or Playwright verification.
  6. Review diffs manually before merge.
  7. Capture reusable learning into AGENTS.md, CLAUDE.md, skills, or templates.
