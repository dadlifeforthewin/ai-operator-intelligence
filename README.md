# AI Operator Intelligence

A polished static GitHub Pages dashboard for daily AI builder intelligence.

## Included

- Premium dashboard homepage
- Daily archive page
- Search and topic filtering
- Signal cards
- Trend tracker
- Operator playbook
- GitHub Pages deployment workflow

## Publish on GitHub Pages

1. Create a new GitHub repository named `ai-operator-intelligence`.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **GitHub Actions**.
5. Push to `main`.
6. GitHub Actions will publish the site.

Your URL will usually be:

```txt
https://YOUR-GITHUB-USERNAME.github.io/ai-operator-intelligence/
```

## Daily workflow later

1. Generate daily briefing.
2. Save as `archive/YYYY-MM-DD.html`.
3. Update `index.html` latest signal cards.
4. Commit and push.
5. GitHub Pages publishes automatically.
6. Email sends a short summary with the dashboard link.
